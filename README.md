# Rock Paper Scissors Lizard Spock

Play Rock, Paper, Scissors, Lizard, Spock against the computer.

## Rules

-   Scissors beats Paper
-   Paper beats Rock
-   Rock beats Lizard
-   Lizard beats Spock
-   Spock beats Scissors
-   Scissors beats Lizard
-   Paper beats Spock
-   Rock beats Scissors
-   Lizard beats Paper
-   Spock beats Rock
